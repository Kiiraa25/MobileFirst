La structure minimale du DOM

Dom veut dire Document Object Model

Voici la structure minimale du DOM

Je déclare le DOCTYPE html

J'ouvre un le document html : <html>

Je crée un entête : <head>

J'y mets un titre qui va s'afficher dans l'onglet du navigateur: <title>

je crée un corps de page : <body>

je referme bien toutes mes balises sans qu'elle se croisent !
